Tsoding Base

How do I contribute

If you have access to Tsoding's discord server you can write to #TsodingBase thread and ask what needs to be done. At this moment VODs need to be tagged, rules on that and untagged VODs are listed on home page

If you don't have access to Tsoding's discord you can dm me on discord (tag @linux_seller)

Contribution process

VODs db contribution

  1. Ask @linux_seller what VODs are not processed (most recent info should be displayed on homepage, but it is better to ask)
  2. Clone repo of homepage containing tags.json
  3. Update ONLY tags.json according to rules and create a pull request
  4. Wait for review

Code contribution

If you are trusted person in Tsoding's discord you are allowed to become part of TsodingBase-community organisation on github, where you can create repo to implement your idea

If you are not a trusted person, dm @linux_seller on discord for further discussion

Idea reporting

If you have any ideas you can make a github issue on Guide Lines repo
